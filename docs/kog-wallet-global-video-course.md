当今世界，越来越多商家正在面对同样的问题：没有稳定客户、广告成本越来越高、现金流越来越紧、会员经营越来越困难。很多商家并不是没有好产品，也不是没有服务热情，而是缺少一个可以持续带来客户、连接社区、建立信任的生态系统。

有没有一种方法，能够让商家彼此合作，让消费者在合作商家之间流动，让 AI OPC 帮助商家自动获客、自动经营会员、自动制作内容，并共同形成新的商业生态？

欢迎来到 KOG Wallet。

KOG Wallet 不是银行，不是投资平台，也不是一个单纯的支付工具。KOG Wallet 的核心使命，是帮助商家、会员与 AI OPC 共同形成全球闭环经济圈。

欢迎来到第三集。本集我们要完成 KOG Wallet 注册流程。

# EP05 如何使用 QR Code 完成商家交易

**建议时长：8–10 分钟**

## 本集目标

说明扫描 QR Code、确认商家、选择优惠券、完成付款或折抵、查看交易纪录的完整流程。

## 逐字旁白稿（繁体中文）

在 KOG Wallet 生态中，QR Code 是连接会员与商家的关键入口。

当你来到合作商家消费时，请先确认店内展示的 KOG Wallet QR Code 是否属于该商家。你可以打开 KOG Wallet，点击扫描，扫描商家的 QR Code。系统会显示商家名称、地址、交易项目或金额输入页面。

确认商家资讯正确后，选择可用优惠券。系统会自动显示折抵规则、折扣金额与最终应付金额。请在确认无误后点击确认。

如果扫描失败，请检查手机相机权限、网络连接、QR Code 是否清晰，或请商家重新显示 QR Code。

QR Code 的意义不只是付款或折抵，更重要的是把一次消费变成可追踪、可经营、可再营销的会员关系。

# EP06 如何成为合作商家

**建议时长：10–12 分钟**

## 本集目标

说明商家申请、资料提交、审核、上线、生成 QR Code 与开始营业的流程。

## 逐字旁白稿（繁体中文）

KOG Wallet 的核心，是把真实商家连接成全球商家联盟。本集我们要说明如何成为合作商家。

商家首先需要进入申请入口，填写基本资料，包括店名、行业类别、地址、联络人、电话、Email、营业时间、服务项目与商家介绍。为了保护会员与生态品质，商家资料需要经过审核。

审核重点包括：商家是否真实存在、服务项目是否清楚、联络方式是否有效、是否愿意遵守合作规则，以及是否能提供会员可理解的优惠或服务。

审核通过后，商家会取得后台权限，并生成专属 QR Code。这个 QR Code 可以放在柜台、菜单、海报、收银台、网站、社群媒体与活动现场。

上线后，商家可以开始接收会员扫码、发放优惠券、查看交易纪录、经营客户资料，并透过 AI OPC 协助制作内容、设计活动与追踪数据。

成为合作商家的关键不是单纯加入系统，而是愿意参与联盟合作。餐厅可以与咖啡店合作，旅馆可以与诊所合作，学校可以与教会合作，律师事务所可以与 AI 公司合作。生态越真实，会员价值越高。
